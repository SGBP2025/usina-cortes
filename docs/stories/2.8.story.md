## Story
**As a** usuário  
**I want** ver todos os meus processamentos anteriores e fazer download dos clipes  
**So that** não perca trabalho já feito

## Acceptance Criteria
- [ ] AC1: Página `/dashboard` (home) exibe lista dos últimos 10 jobs com status, data e número de clipes
- [ ] AC2: Página `/dashboard/clips` exibe todos os clipes gerados com preview e download
- [ ] AC3: Cada ClipCard exibe: duração, título YouTube, descrições TikTok/Instagram, botão download
- [ ] AC4: Download via URL assinada do Supabase Storage (expiração 1h)
- [ ] AC5: Estado vazio quando nenhum job/clip existe
- [ ] AC6: Links para `/dashboard/jobs/[id]` de cada job na lista

## Tasks

- [ ] T2: Atualizar `app/(dashboard)/clips/page.tsx` com galeria de clipes

## Dev Notes
URLs assinadas via `supabase.storage.from('clips').createSignedUrl(path, 3600)`. Dados carregados server-side via Server Components para evitar flash de conteúdo.
